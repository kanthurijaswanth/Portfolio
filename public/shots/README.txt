Put your real project screenshots in THIS folder.

For the LMS, save your screenshot here as exactly:
    lms.png

(So the full path is:  public/shots/lms.png )

The site is already wired to use it — once the file is here, the LMS project
card automatically shows your real screenshot instead of the animated mockup.
Until then, the animated mockup is shown (nothing looks broken).

You can do the same for other projects by setting `visual.shot` in src/data.js,
e.g. "./shots/music.png" or "./shots/ticketresell.png".
